# Router Manager Frontend

React + Vite tabanlı modern admin paneli.

## Kurulum

```bash
# Bağımlılıkları yükle
npm install

# .env dosyasını oluştur
cp .env.example .env

# Development server'ı başlat
npm run dev
```

## Build

```bash
# Production build
npm run build

# Build'i önizle
npm run preview
```

## Yapılandırma

`.env` dosyasında `VITE_API_BASE_URL` değişkenini backend API URL'inize göre ayarlayın.
